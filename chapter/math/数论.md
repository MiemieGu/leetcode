bool is_prime(int x) {
    if (x < 2) return false;
    for (int i = 2; i <= x / i; i ++) // 枚举从 i 到 x/i 是否能整除 x
        if (x % i == 0) return false;
    return true;
}

## 题目：尾部的零

计算出n阶乘中尾部零的个数

枚举找规律：

1!  1 0
2！ 1*2 0
3！1*2*3 0
...
5！ 1*2*3*4*5 1
...
10! 1*2*..*5*...*10 2个5 2
...
15! 三个5:5,10,15   3
...
20! 4个5  4
...
25! 5,10,15,20,25,其中25有两个质因子5，6

求阶乘中质因子5的个数

125

5 -- 25
25 -- 5
125 -- 1

130 -- 26

```cpp
class Solution {
public:
    int trailingZeroes(int n) {
        long long ans = 0;
        for (int i = 1; pow(5, i) <= n; i++){
            ans += n / pow(5, i);
        }
        return ans;
    }
};
```

## 题目：找数字II

给出两个数字 A 和 B，找出 A 与 B 之间所有的数（包括 A 和 B），使得这些数满足是 3 的倍数，返回这些数的和。

1 <= A <= B <= 10000000

等差数列求和公式: `（首项+末项）*项数 /2 `

```cpp
class Solution {
public:
    int getSum(int A, int B) {
        while (A % 3!=0) A++;
        while (B%3 != 0) B--;
        int sum = (A + B)* ((B-A) /3 + 1)/2;
        return sum;
    }
};
```

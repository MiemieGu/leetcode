给定一个链表，返回链表开始入环的第一个节点。 如果链表无环，则返回 null。

示例 1：

输入：head = [3,2,0,-4], pos = 1
输出：tail connects to node index 1
解释：链表中有一个环，其尾部连接到第二个节点。

```javascript
var detectCycle = function (head) {
    let dataMap = new Set()
    while (head) {
        if (dataMap.has(head)) {
            return head
        }
        dataMap.add(head)
        head = head.next
    }
    return null
};
```

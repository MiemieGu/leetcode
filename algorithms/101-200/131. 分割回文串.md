给定一个字符串 s，将 s 分割成一些子串，使每个子串都是回文串。

返回 s 所有可能的分割方案。

示例:

输入: "aab"
输出:
[
  ["aa","b"],
  ["a","a","b"]
]

**子问题**

动态规划判断是否回文

```cgo
// 是否回文串；动态规划思路
// 对于字符串 s。
// 用 dp[i][j] 表示 s[i，j] 是否是回文串。
// 然后有 dp[i][j] = s[i] == s[j] && dp[i+1][j-1]
func IsPalindrome(s string) [][]bool {
	dp := make([][]bool, len(s))
	for i := 0; i < len(s); i++ {
		dp[i] = make([]bool, len(s))
	}
	for k := 1; k <= len(s); k++ { // 长度k
		for i := 0; i <= len(s)-k; i++ {
			j := i + k - 1;
			dp[i][j] = s[i] == s[j] && (k < 3 || dp[i+1][j-1]);
		}
	}
	return dp
}
```

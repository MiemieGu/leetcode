#### [136. 只出现一次的数字](https://leetcode-cn.com/problems/single-number/)



#### 解题思路

异或运算

1. 任何数与0异或结果都为他本身
2. 任何两个相同的数异或结果都为0
3. 异或运算满足交换律和结合律

### 代码

cpp

```cpp
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int ans = 0;
        for (auto n : nums){
            ans ^= n;
        }
        return ans;
    }
};
```

javascript

```javascript
var singleNumber = function(nums) {
    let ans = nums[0]
    for (let i =1; i< nums.length; i++) ans ^= nums[i]
    return ans
};
```

```java
class Solution {
    public int singleNumber(int[] nums) {
        int ans = 0;
        for (int x: nums) {
            ans ^= x;
        }
        return ans;
    }
}
```

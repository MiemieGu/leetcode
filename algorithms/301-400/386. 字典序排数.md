给定一个整数 n, 返回从 1 到 n 的字典顺序。

例如，

给定 n = 13，返回 [1,10,11,12,13,2,3,4,5,6,7,8,9] 。

请尽可能的优化算法的时间复杂度和空间复杂度。 输入的数据 n 小于等于 5,000,000。

* 字典序其实就是字符串排序

```javascript
var lexicalOrder = function (n) {
    let A = new Array(n)
    for (let i = 0; i < n; i++) {
        A[i] = i + 1 + ''
    }
    return A.sort()
};
```

* 深度优先搜索之先序遍历

```javascript
var lexicalOrder = function (n) {
    let ans = []
    function dfs(num) {
        if (num <= n) ans.push(num)
        else return // 剪枝
        for (let i = 0; i <= 9; i++) {
            dfs(num * 10 + i)
        }
    }
    for (let i = 1; i < 10; i++) dfs(i)
    return ans
};
```

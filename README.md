## 特色功能

### 自动生成题目列表

运行脚本`node script/genTOC.js`，程序会获取本地浏览器cookie，并请求得到解题列表，自动生成目录文件`TOC-By-ID.md`和`TOC-By-Tag.md`

### 动态图解制作

图解制作使用开源工具`graphviz`，基于dot语言生成图片，使用`GraphicMagick`进行图片处理，自动生成动图

脚本使用`node script/genPic.js`

## 二分法算法思想

将区间分为左右两边，判断中点，确定答案在哪一边，每次缩小一半，直到得到最终答案。

70%的二分都是跟单调性有关，有单调性的题目一般都可以二分。
95%的题目存在两段性的性质。一部分满足，一部分不满足。

## 二分算法模板

将区间[l, r]划分成[l, mid]和[mid + 1, r]，
其更新操作是r = mid或者l = mid + 1;
计算mid时不需要加1。

```javascript
function divide(l, r) {
    while (l < r) {
        let mid = l + r >> 1
        if (check(mid)) r = mid
        else l = mid + 1
    }
    return l
}
```

# 图的遍历

图的遍历一般有dfs和bfs两种方式

注意事项：

1. 区别有环，无环
2. 判断当前节点是否已得到最优解
3. 判断是否可以剪枝
4. 弧头到弧尾还是弧尾到弧头，遍历顺序如何选择
